# General Project Rules

See the root AGENTS.md for the main guidance.

Additional notes:
- This is an agent capabilities playground. When adding new skills, agents, or personas, try to keep both a rich Grok-native version (in .grok/) and a portable markdown version (in agents/).
- Prefer structures that other tools (Claude, Cursor, Copilot) can also benefit from.
- Use the review-component skill liberally when touching UI code.
