# Apollo Connectors (Portable)

Portable guide for Apollo Connectors to integrate REST into GraphQL.

## In This Monorepo
- Use to connect external REST to the apps/api GraphQL backend.
- Then staff/public UIs (and agents via MCP) can query unified data.
- Validate with rover, test via portless.
- Pair with update-graphql, graphql-schema, apollo-mcp-server.

See the full adapted version at `.grok/skills/apollo-connectors/SKILL.md` for schema template, rules, and monorepo ties.
