# Portable Skills

This directory contains tool-agnostic versions of the skills defined in this repository.

These are clean Markdown descriptions of repeatable procedures. They are designed to be:

- Loaded as context by Claude, Cursor, GitHub Copilot, Aider, Continue, etc.
- Referenced from AGENTS.md, CLAUDE.md, .cursorrules, or copilot-instructions.
- Easy for humans to read and follow.

The "source of truth" rich versions (with Grok frontmatter, tool hints, etc.) live in `.grok/skills/`.

## Current Portable Skills (including Apollo GraphQL)

See the individual .md files here (e.g. `review-component.md`, `implement-feature.md`, `add-antd-component.md`, `update-graphql.md`, `apollo-server.md`, `apollo-client.md`, `graphql-schema.md`, `graphql-operations.md`, `apollo-mcp-server.md`, `rover.md`, `apollo-federation.md`, `apollo-connectors.md`, `skill-creator.md`, `apollo-router.md`, `turborepo` (full directory with SKILL.md + references), etc.).

For the absolute latest or Grok-optimized versions, refer to the corresponding file under `.grok/skills/`.
